Detailed Description

This is the packet that is sent universally ALWAYS 18 bytes wide (16 bytes for data (payload) and 2 bytes for header) Without #pragma pack(1), this structure is actually 24 bytes, but the contents add to 18 (cause of padding)

header (2 Bytes) [3:0]: payloadType - sets the type of data in the packet's payload [4] : isPayloadValid - marks if the data contained in the payload is valid or not (ex: did a sensor reading return true or not) [5] : extraBool - an extra bool to use as needed [7:6]: dest address - the address of the destination radio to send the packet to [9:8]: src address - the address of the source radio (the one that is sending it) [10] : expectACK - a bool letting the receiver know whether the transmitter wants an ACK packet back or if it should just assume data was received
