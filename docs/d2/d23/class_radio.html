Detailed Description

The abstract class to use when creating new Radio classes Have all derived Radio classes inherit this class. All these methods, structs, and variables will be available in it. This standardizes all Radio modules we use.

Additionally, have all functions that need a Radio module take this class as a param. This way any Radio class that inherits this one will be able to be passed in

Parameters
packetThe packet whose data will be set
tft_screenThe object for the TFT ILI9341 screen that the message will be displayed on
tempMessageLine1The temporary msg to display on line1 while it waits for a package to come in. ex: "waiting for confirmation of landing"
tempMessageLine2Optional. The temporary msg to display on line2 while it waits for a package to come in. If param is ignored will print "Block for: infinity"
Returns
True if a packet was received. False if a packet was not received (the timeout was hit)
