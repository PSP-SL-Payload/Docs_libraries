Detailed Description

The abstract class to use when creating new GPS classes Have all derived GPS classes inherit this class. All these methods, structs, and variables will be available in it. This standardizes all GPS modules we use.

Additionally, have all functions that need a GPS module take this class as a param. This way any GPS class that inherits this one will be able to be passed in
