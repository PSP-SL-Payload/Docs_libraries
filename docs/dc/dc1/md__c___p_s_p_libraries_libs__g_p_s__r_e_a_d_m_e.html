PSP Libraries
What is the GPS abstract class?

This class is an abstract class, meaning that is kind of like a template. It isn't exactly a template, because that's a specific type of class in C++ but we're using it as a template for other GPS modules. Every single function, struct, variable, etc in this class will be available in its child classes.

When should I use this class?

Having functions that need a GPS

Any function that needs a GPS module should take this GPS class as a parameter.

For example, let's say you have a method called bool IsPayloadStationary(GPS gps).

Notice here I have the parameter type as GPS. This means any child class of GPS can be passed in here. We could pass in an object of GPS_SAM_M8Q, GPS_TVY7, etc as long as the class that gets passed in is a child class of GPS.

Example:

{C++}
foo()
{
// we make an GPS_SAM_M8Q object (a child of GPS)
GPS_SAM_M8Q gps;
...
// we call IsPayloadStationary (notice that we pass in a GPS_SAM_M8Q, NOT a GPS object)
IsPayloadStationary(gps);
...
}

Making child classes

Whenever you have a new GPS module, have the child class you make for it inherit this class.

For example, let's say I'm making a new child class for the TVY7 gps module.

  1. Start by creating a new child class called GPS_TVY7.
  2. Have it inherit this class, like so: class GPS_TVY7 : public GPS.

Any abstract function in the GPS class needs to implemented in the GPS_TVY7 class. You can find these by looking at all the fuctions of this general format in the header file virtual void foo() = 0;

Now your child class can be used in the control loop while changing virtually nothing because its functionality is the same as other GPS classes.

Example Test Code (change pins and such accordingly)

{C++}
#include <Arduino.h>
#include childClass
// TODO: change childClass to the respective child class, ex: GPS_SAM_M8Q
childClass gps;
bool wasLocationFound;
void setup()
{
Serial.begin(9600);
gps.Init();
// attempt to find the location
wasLocationFound = BlockUntilLocationAquired(GPS_TIME_TO_WAIT_FOR_LOCATION_MS);
}
void loop()
{
gpsLocation location;
gpsDateTime dateTime;
if (wasLocationFound)
{
GetLocation(location);
GetDateTime(dateTime);
GPS::PrintLocation(location);
GPS::PrintDateTime(dateTime);
}
else
{
Serial.println("Location not found...");
}
}
