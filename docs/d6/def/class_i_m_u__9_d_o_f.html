Detailed Description

The abstract class to use when creating new 9 Degrees of Freedom OR IMU classes Have all derived 9 DoF IMU classes inherit this class. All these methods, structs, and variables will be available in it. This standardizes all IMU modules we use.

Additionally, have all functions that need a 9 DoF IMU module take this class as a param. This way any 9 DoF IMU class that inherits this one will be able to be passed in
