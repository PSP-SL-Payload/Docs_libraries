enum MissionStageDetection::StageOfMission

All the defined stages of flight. Feel free to add more as needed for each competition. Make sure they follow the EXACT order of operations that will be executed. This is because some of the blocking methods have 'skipping' features that will skip it if the actual stage is past the respective function.
