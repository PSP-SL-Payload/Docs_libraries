Detailed Description

The abstract class to use when creating new Altimeter classes Have all derived Altimeter classes inherit this class. All these methods, structs, and variables will be available in it. This standardizes all Altimeter modules we use.

Additionally, have all functions that need a Altimeter module take this class as a param. This way any Altimeter class that inherits this one will be able to be passed in
