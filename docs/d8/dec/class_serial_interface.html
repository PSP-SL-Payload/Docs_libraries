SerialInterface::SerialPacketNode * SerialInterface::WaitForPacket ( uint32_t  packetID,
uint32_t  timeoutMilliseconds = TIMEOUT_INFINITY 
)

Blocks and waits for a packet to be received over the serial COM port that matches packetID.

Parameters
packetIDThe packet ID to wait for, any other packets read before it will be discarded.
timeoutMillisecondsThe duration to wait before timing out if the packet has not been read yet. It's defaulted to TIMEOUT_INFINITY.
Returns
Returns an allocated pointer to a SerialPacketNode, or TIMEOUT if it timed out.
