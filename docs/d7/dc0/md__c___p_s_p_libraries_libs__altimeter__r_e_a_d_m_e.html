What is the Altimeter abstract class?

This class is an abstract class, meaning that is kind of like a template. It isn't exactly a template, because that's a specific type of class in C++ but we're using it as a template for other Altimeter modules. Every single function, struct, variable, etc in this class will be available in its child classes.

When should I use this class?

Having functions that need an Altimeter

Any function that needs an Altimeter module should take this Altimeter class as a parameter.

For example, let's say you have a method called float DetectApogee(Altimeter altimeter).

Notice here I have the parameter type as Altimeter. This means any child class of Altimeter can be passed in here. We could pass in an object of Altimeter_BMP280, Altimeter_BMP390L, etc as long as the class that gets passed in is a child class of Altimeter.

Example:

{C++}
foo()
{
// we make an Altimeter_BMP280 object (a child of Altimeter)
Altimeter_BMP280 altimeter;
...
// we call DetectApogee (notice that we pass in an Altimeter_BMP280, NOT an Altimeter object)
DetectApogee(altimeter);
...
}

Making child classes

Whenever you have a new Altimeter module, have the child class you make for it inherit this class.

For example, let's say I'm making a new child class for the BMP390L altimeter module.

  1. Start by creating a new child class called Altimeter_BMP390L.
  2. Have it inherit this class, like so: class Altimeter_BMP390L : public Altimeter.

Any abstract function in the Altimeter class needs to implemented in the Altimeter_BMP390L class. You can find these by looking at all the fuctions of this general format in the header file virtual void foo() = 0;

Now your child class can be used in the control loop while changing virtually nothing because its functionality is the same as other Altimeter classes.

Example Test Code (change pins and such accordingly)

{C++}
#include <Arduino.h>
#include childClass
// TODO: change childClass to the respective child class, ex: Altimeter_BMP280
childClass altimeter;
void setup()
{
Serial.begin(9600);
altimeter.Init();
}
void loop()
{
Altimeter::AltimeterReadings altReadings;
if (altimeter.GetAltimeterReadings(altReadings))
{
Altimeter::PrintAllReadings(altReadings);
}
delay(400);
}
