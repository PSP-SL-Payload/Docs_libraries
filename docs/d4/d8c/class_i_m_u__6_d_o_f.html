Detailed Description

The abstract class to use when creating new 6 Degrees of Freedom OR 9 Degrees of Freedom IMU classes Have all derived IMU classes inherit this class. All these methods, structs, and variables will be available in it. This standardizes all IMU modules we use.

Additionally, have all functions that need a IMU module take this class as a param. This way any IMU class that inherits this one will be able to be passed in
